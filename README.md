
# 🚀 TipFlow

TipFlow is a web application designed for waiters to track tips, income, and daily performance in a simple and scalable way.

## 🧠 Objective

Build a real-world tool that helps people in the restaurant industry:

- Track daily tips
- Analyze income
- Improve financial performance

## 🛠️ Tech Stack

- Python (Flask)
- SQLite (for now)
- SQLAlchemy
- REST API
